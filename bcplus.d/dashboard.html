Greetings commander! Please choose a topic from the blue buttons above…

Topics

Ships
This topic shows all BC+ learned about your ships from your journal files.
Travel
Gives you an overview and let's you define end edit designations in the milky way. Estimates about travel duration an number of jumps are made based on the maximum collected jump range of the selected ship and the jump statistics from the most recent jumps.
Bodies
Lists all the bodies scanned in the current system along with materials and their availability.
Materials
Gives a comprehensive overview of how many of which materials you have. And how many of those you need, either for a manually set demand or for synthesis you plan.
Synthesis
Here you can plan how many of which recipes for synthesis you need.

General

  • If you see this keyboard icon , text before the icon is highlighted when your mouse hovers over it. With a single click the highlighted text is send as keyboard input on the computer running BC+. This is particularly useful to send system names to the input box of the galaxy map.
  • BC+ is currently not connected to any other E:D player platforms like EDDN etc. This is planned for the future. However interaction with those community sites and—even more—with the companion API involves handling of sensitive user credentials. This needs some security build into BC+: Encrypted storage of those credentials as well as using HTTPS for the Web GUI. Integration will not be implemented before security is in place.
